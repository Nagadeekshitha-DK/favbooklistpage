# favbooklistpage link:https://listoffavbooks.ccbp.tech/
A simple and responsive Book Store Website built using HTML, CSS, and Bootstrap. This project displays a home page with popular and recommended books, along with individual detail pages for each book.
Features:
Home page with Popular and Recommended books
Separate detail pages for each book
Navigation between pages using buttons
styled using custom CSS and Google Fonts
Responsive layout using Bootstrap
